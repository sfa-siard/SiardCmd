Downloading the Binaries of the Project

This user's guide explains the downloading and using of the binaries of the project. For instructions on downloading and building the sources of the project, please consult the Developers' Manual.

The binaries of the project are stored under "Releases". They consist of the JAR file zip64.jar for automatic download by other projects and the full binary release zip64-<v>.<r>.<b>.zip. The full binary release contains a copy of the JAR file.

In order to make use of the binaries the full binary release must be copied and unzipped locally.

Folders of the Binary Release

The binary release has the following folder structure:

doc/
documentation of the project
doc/javadoc/
javadoc documentation of published APIs of the project
doc/licenses/
license files for third-party code
doc/manual/
manuals documenting the project
doc/manual/users/
documentation for users of the binaries (in HTML5 format)
doc/manual/assets/
common files for all manual pages
doc/specifications
specifications implemented by the project
etc/
files for run-time configuration (e.g. logging.properties)
lib/
zip64.jar as well as binaries of other projects and third-party code
LICENSE.txt
license of the project
README.txt
very short description of the project
RELEASE.txt
documentation of release changes
