SiardCmd - Users' Manual - Purpose

Purpose of the Project

SiardCmd was developed as part of the SIARD project. It is used by the SiardGui implementation of the SIARD project in order to download and upload SIARD files adhering to the standardized SIARD Format 2.1 from and to an relational database instance.
